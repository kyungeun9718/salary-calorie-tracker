# 실시간 월급 & 칼로리 트래커

"1초에 얼마나 벌까?", "칼로리는 얼마나 태울까?"를 실시간으로 보여주는 재미용 웹앱입니다.

설치나 서버 없이 `index.html` 파일 하나로 동작합니다.

## 사용법

`index.html`을 브라우저(크롬 권장)로 열면 됩니다.

1. 월급과 급여 계산 모드를 선택합니다.
   - **근무시간만**: 출근~퇴근 시간에만 카운트
   - **24시간 항상**: 하루 종일 계속 누적
   - **수동 시작/종료**: 버튼을 눌러 직접 근무 시작/종료
2. 성별, 나이, 키, 몸무게, 활동 수준을 입력합니다.
3. "실시간으로 보기 시작"을 누르면 급여와 칼로리가 실시간으로 올라갑니다.

## 계산 방식

- **급여**: 입력한 월급과 근무 조건(근무시간, 월 근무일수 등)을 기준으로 초당 급여를 계산해 누적 표시
- **칼로리**: [Mifflin-St Jeor 공식](https://en.wikipedia.org/wiki/Basal_metabolic_rate)으로 기초대사량(BMR)을 구하고, 활동 계수를 곱해 하루 총 소모 칼로리(TDEE)를 계산 → 초당 소모량으로 환산해 자정부터 누적 표시
- 두 값의 비율로 "1kcal 태우는 동안 OO원 벌어요" 문구도 함께 보여줍니다

## 참고

- 입력값은 브라우저 localStorage에 저장되어 새로고침해도 유지됩니다 (저장이 막힌 환경에서도 실시간 표시 자체는 계속 동작합니다).
- 칼로리 100kcal 단위를 넘을 때마다 레벨업 연출과 효과음이 재생됩니다.
